# Conecta Frontend

O **Conecta** é o MVP de uma plataforma web desenhada para organizar e dar visibilidade ao mercado de serviços informais. O sistema atua como uma vitrine digital, conectando trabalhadores autônomos a potenciais clientes através de um catálogo interativo.

---

## Como instalar as dependências

Acesse o diretório do projeto no seu terminal e execute o comando abaixo para instalar os pacotes necessários:

```bash
npm install
```

## Como subir a aplicação 
Execute o comando abaixo 
```bash
npm run dev
```
